# MinimaUnit - Smart Converter 公開チェックリスト

## 自動チェック

```bash
npm run release:check
```

## 画面チェック

- [ ] ヘッダーのロゴがデスクトップ / スマホで崩れない
- [ ] ツール切替が全機能で表示される
- [ ] 設定 / 履歴 / 案内ドロワーが開く
- [ ] `今日の日付` と上部メッセージが自然に読める

## 機能チェック

- [ ] 年齢・期間で年号・西暦・経過期間が出る
- [ ] 長さ / 重さ / 面積 / 体積 / 温度 / 速度が入力即時反映する
- [ ] 通貨で基準日・最終更新・レート更新が表示される
- [ ] 結果コピーが主要画面で動く

## スマホ確認

- [ ] 横スクロールが出ない
- [ ] 数値入力でテンキーが出やすい
- [ ] ボタン同士が詰まりすぎていない
- [ ] ヘッダー右上のアイコンが押しやすい

## ブランド確認

- [ ] ブラウザタイトルが `MinimaUnit - Smart Converter`
- [ ] favicon が新ロゴになっている
- [ ] OGP カードが新ブランドになっている
- [ ] manifest の `name` / `short_name` が新ブランドになっている

## 公開前の最終確認

- [ ] Cloudflare Pages の Build command が `npm run build`
- [ ] Output directory が `dist`
- [ ] `NODE_VERSION=20`
- [ ] カスタムドメインを使う場合は `canonical` と `og:url` の追加方針を確認済み
